# Engineering Instructions
- When I ask you to "Check Modem for what's next," use the GitHub/Linear tool to find the latest issue tagged 'triage'.
- Read the 'Context' section of the issue (which contains the Modem summary).
- Implement the fix in a new branch named `modem/issue-number`.
- For game polish and fidelity work involving timing, motion, audio, pacing, or transitions, default to measured reference analysis first.
- Use archived gameplay video, extracted reference clips, waveforms, contact sheets, and generated timing-analysis artifacts before subjective tuning.
- Prefer preserving full canonical reference phrases and widening game windows to fit them when feasible; only use shortened runtime-safe excerpts when measurement shows a full clip cannot fit, and document the reason.
