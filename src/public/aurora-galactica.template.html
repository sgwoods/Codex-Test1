Current state

Primary goal

Keep Platinum Release 1 stable while we separate platform and application documentation, close the small Aurora trust-fix cluster, and prepare the second-game seam without rushing it into production.

What is already solid

Platinum now owns the shell, picker, hosted lane model, and release framing, while Aurora remains a playable shipped pack with the main release documentation, dashboard, and player guide generated from maintained source docs.

Current blocker

The biggest remaining risk is not broad architecture uncertainty anymore. It is making sure the fast-follow Aurora fixes, the hosted docs, and the platform/application boundary all stay aligned across localhost, hosted /dev, hosted /beta, and hosted /production.
