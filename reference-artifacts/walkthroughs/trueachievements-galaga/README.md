# TrueAchievements Galaga Walkthrough Notes

Source URL:
- https://www.trueachievements.com/game/Galaga/walkthrough

Role in this project:
- secondary reference, not a primary rules source
- useful for later-stage content patterns, transform-enemy cadence, and player-facing progression ideas
- not authoritative for original arcade Galaga behavior, especially where it discusses modern port conveniences such as options tweaks or starting from the highest unlocked stage

Useful observations:
- The guide distinguishes capture outcomes clearly:
  - destroying the captured fighter
  - destroying the commander ship to regain a dual fighter
- It describes recurring transform enemy bands:
  - Scorpions in stages 4-6
  - Stingrays in stages 8-10
  - Galbosses in stages 12-14
  - repeating thereafter
- It calls out challenge-stage enemy variants appearing on specific later challenge stages:
  - Dragonfly around challenge stage 11
  - Mosquito around challenge stage 19
- It reinforces that later-stage content variety is an important part of the Galaga experience, not just faster versions of the same early-stage enemies.

What not to copy directly:
- changing lives / extend settings for easier completion
- stage-select restart from the highest unlocked stage
- achievement-specific routing decisions that are about the port, not the arcade design

Implications for our tribute:
- later-stage enemy variety is still underdeveloped in our game
- transform-style enemy families and stage-banded content could add a lot of authenticity after core balance is stable
- capture / rescue remains central enough that it should keep a dedicated fidelity workstream
