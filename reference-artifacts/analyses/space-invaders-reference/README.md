# Space Invaders Reference Lineage

This analysis lane holds committed, reviewable reference artifacts for the
planned `Windigo Invaders` game line.

## Current Status

- lineage: `space-invaders-reference`
- stage: `artifact-collection-and-analysis`
- current promoted source count: `6`
- primary use: establish a Platinum-focused ingestion and instantiation plan
  from real reference evidence before designing the runtime

The most useful next artifact families for this lineage are now:

## Newly Added Intake Value

The new preserved rules/techdocs package adds:

- readable row-score values and a full-screen score total
- bonus-base and bases-per-game setup truth
- derivative gameplay/rules prose that is good enough to stop guessing at the
  first Windigo scoring and lives policy
- service/manual bundles that can later yield stronger still-image and page
  captures

Primary anchor:

- `reference-artifacts/preserved-sources/space-invaders-rules-techdocs-2026-05-31/README.md`

The new ClassicGaming sound pack adds a second strong anchor:

- `reference-artifacts/preserved-sources/space-invaders-classicgaming-sounds-2026-06-01/README.md`

It means the sound-inventory gap is no longer generic. We now have named march,
UFO, shot, hit, and explosion sounds in repo, so the remaining audio hunt
should focus on start / credit / player-loss truth and on gameplay-context
windows that show those sounds under real march and bunker pressure.

The new `space-invaders-absoluteknave-history-review` page is not timing truth
either, but it is unusually explicit about:

- why the speeding four-note march matters to the felt tension
- how the one-shot projectile rule changes late-wave pressure
- why column-chop strategy matters more than naive row clearing
- how the original left-button / right-button cabinet input changes the feel

That makes it useful for Windigo interpretation and control-model caution, but
it does not materially reduce the measured artifact queue.

The new `space-invaders-arcade-museum` package is useful for a different lane:
cabinet, control-model, and display grounding plus image-lead discovery. It
gives us release and regional distribution metadata, `2-way` plus `1-fire`
control truth, amplified `mono` sound, and the important note that original
Space Invaders presentation was `black and white with a color overlay` rather
than generic full-color sprite display. It also exposes concrete marquee,
cabinet, bezel, control-panel, PCB, screenshot, flyer, and manual targets.
That strengthens Windigo documentary grounding and later presentation review,
but it should not be treated as gameplay-timing truth.

The new `space-invaders-strategywiki-guide-walkthrough-sprite-bundle` package
is now the strongest repo-owned flyer and core gameplay-silhouette lane for
Windigo. It gives us a preserved flyer plus the saucer, the three invader-row
families, and the player cannon along with guide/gameplay/walkthrough prose.
That means the still-image queue is no longer about generic flyer or
invader-family hunting. The more important remaining visual gaps are bunker
references and cabinet-facing marquee, bezel, control-panel, title, and
gameplay surfaces.
