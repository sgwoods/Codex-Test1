## Current Role

This accessioned pack is the strongest repo-owned Galaxian-family source for
named isolated core cues.

It is especially useful for:

- `Credit Sound`
- `Start Game`
- `Shoot`
- `Fighter Loss`
- `Extra Life`
- `Flying Sound`
- `Hit Enemy`
- `Hit Boss`

## What It Changes

This source largely removes the need to infer those eight cue families from the
older static-card Galaxians sound reel.

That means the older reel is no longer the best path for generic core-cue
identification. It is now mainly useful for:

- any remaining cue families not present in this FLAC pack
- any longer mixed or sustained regions that do not appear as isolated tracks
- any residual human-listening comparison where the FLAC pack alone is too
  decontextualized

## What It Does Not Replace

This pack does not replace:

- the no-voiceover gameplay clip for exact early-session cue timing
- the ambience background source for recurring pressure / ambient-bed fitting
- long-session gameplay windows for stage-five-and-beyond pressure and failure
  review
- any future gameplay-context comparison that depends on cue overlap or live
  mix balance rather than isolated shapes
