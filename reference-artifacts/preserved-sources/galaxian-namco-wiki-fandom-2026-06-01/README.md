## What This Source Is

This preserved source is a saved `Namco Wiki | Fandom` page for `Galaxian`.

contains both prose context and embedded links to specific Galaxian image

## Current Role

This package is not primary timing truth and not a direct audio source. Its
value is more concrete than the Wikipedia PDF in one important way: it exposes
specific follow-on still/image targets we can preserve next.

It is especially useful for:

- harvesting direct image leads for flyer, cabinet, and alien-family stills
- preserving lineage/context prose from a game-focused Namco fan wiki
- finding game-specific pages for `Galaxian Flagship` and related alien assets
