# Galaga Screen Surface Targets

This directory records derived target notes for Galaga presentation surfaces:
title, in-game, high-score, game-over, cabinet frame, and challenge-result
surfaces. It intentionally avoids committing third-party screenshots until
the project has an explicit provenance and licensing reason to store them.

The current artifact is a partial screen-surface target map. The next useful
step is to pair each target surface with approved screenshots or controlled
capture frames and then score Aurora title, score, wait, popup, and challenge
result surfaces against those references.
