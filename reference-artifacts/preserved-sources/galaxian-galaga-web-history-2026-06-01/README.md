## What This Source Is

This is the official `Galaga Web` history page for `Galaxian` from Bandai
Namco. It is not primary arcade-era documentary evidence in the same sense as
the Midway operator manual, but it is official publisher-hosted summary
material and therefore stronger than generic fan-summary prose.

## Current Role

This page is especially useful because it provides official summary wording for:

- control model
- single-shot firing constraint
- stage-clear / game-over logic
- alien-family naming
- convoy versus charger score values
- boss/escort summary language

## Confirmed High-Value Facts From The Page

- control model:
  - move the `Galaxip` left and right with the stick
  - press the button to shoot beams
  - beams cannot be fired again until the prior shot disappears
- stage logic:
  - clear the stage by defeating all on-screen aliens
  - game ends when all player Galaxips are destroyed
- score summary:
  - `Green Alien`: convoy `30`, charger `60`
  - `Violet Alien`: convoy `40`, charger `80`
  - `Red Alien`: convoy `50`, charger `100`
  - `Boss Alien`: convoy `60`, charger `150` to `800`
- boss behavior summary:
  - bosses act as leaders
  - only two are in a formation
  - bosses are accompanied by red aliens
  - defeating a boss briefly halts attacks

## What This Source Does Not Replace
