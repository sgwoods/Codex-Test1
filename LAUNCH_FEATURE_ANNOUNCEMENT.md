# Aurora Galactica: 1.0 Launch Features

Aurora Galactica is ready to launch as a polished four-stage arcade slice.

What’s in this launch:

- a tuned four-stage fixed-screen shooter loop
- capture, rescue, and dual-fighter play
- challenging stage support
- in-game pilot profile and sign-in flow
- browser-native recent replay viewer
- in-frame help, controls, scores, feedback, and settings surfaces
- shared leaderboard support for production play
- cleaner release lanes with reviewed beta promotion into production

Highlights:

- replay works directly inside the game rather than as a separate external tool
- pilot identity is now a real part of the game surface
- the shell and popup surfaces stay inside the gameplay frame
- release management is now deliberate:
  - development
  - production beta
  - production

Launch note:

- the true `1.0` production leaderboard will begin from a fresh baseline so
  official post-launch scores reflect the released game rather than pre-launch
  tuning history
