## Current Use

This source extends the current Galaxy Guardians / Galaxian reference profile
with a short gameplay session whose biggest value is precise early-session
cabinet audio without commentary.

It is especially useful for:

- start-screen to control-handoff audio timing
- `BONUS GALAXIP FOR 5000 PTS` and early HUD/rack corroboration
- first-shot, hit, player-loss, and game-over sound review
- early stage-1/2 pacing and formation-settle review

It does not yet replace the need for longer stage-five-and-beyond audio scenes,
but it should now become a primary anchor for early-session Guardians audio
scene extraction.
