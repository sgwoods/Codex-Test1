## Current Role

This accessioned source expands the repo-owned Galaxian reference set with a
full 15-wave session that covers the start / score-advance-table presentation,
live play, later-wave pressure, and game-over closure in one uninterrupted run.

It is especially useful for:

- full-session pacing review from first control handoff through final failure
- score-advance-table, `PUSH START BUTTON`, and `BONUS GALAXIP` presentation
  review in the same source as the later gameplay
- stage-five-and-beyond audio/pacing window promotion from a single preserved
  long session
- deeper-play crowding, recovery, and game-over closure review

## What It Does Not Replace

This source is strategically useful, but it does not replace:

- the higher-resolution `nenriki-15-wave-session` source for finer motion and
  geometry review
- the shorter no-voiceover clip for the most compact early-session audio-window
  promotion work

Its biggest value is continuity: one full run from early presentation through
late failure, which means the next ask is to promote named deeper-play windows
from sources we already have instead of hunting generic longplays.
