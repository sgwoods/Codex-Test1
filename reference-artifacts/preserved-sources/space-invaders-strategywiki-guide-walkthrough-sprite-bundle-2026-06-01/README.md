- cannon shape and silhouette
- the three invader-row families
- the saucer

  reopening generic web pages

invader-row stills.

## What It Does Not Replace

This source does not replace:

- original score-table or manual-page visual truth
- marquee, bezel, control-panel, and title/gameplay cabinet-facing stills
- higher-resolution gameplay windows
- missing start, credit, and player-loss sound truth
- bunker still references
- event-labeled timing windows

Its main value is flyer and sprite-family identity plus supporting walkthrough
context, not timing or live audio truth.
