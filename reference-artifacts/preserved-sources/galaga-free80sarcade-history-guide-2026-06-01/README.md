## Current Role

This preserved page is a supporting Galaga history / strategy / browser-play
artifact, not primary timing truth.

It is most useful for:

- prose descriptions of capture / rescue flow
- entrance-pattern framing
- challenging-stage positioning notes, especially the center-line alignment
  advice for the first challenge stage
- explicit anti-fire exploit explanation, including the "leave one or two bees
  on the far left and wait" setup
- transform-progression summary, including the scorpion / Bosconian / Galaxian
  flagship progression and bonus values
- later-stage pressure explanation, especially the warning that post-wrap speed
  becomes much more dangerous when the screen is still crowded
- historical player-strategy guidance that can sharpen design-grounding notes

Its main value is explicit strategy prose and player-facing explanation, not
authoritative frame or cue timing.
