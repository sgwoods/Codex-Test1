## Current Role

This accessioned pack is the strongest repo-owned named Space Invaders sound
source for the core low-level event vocabulary.

It is especially useful for:

- UFO movement bed
- four march-step cadence cues
- player shot
- invader hit / kill
- explosion

## What It Changes

This source largely closes the generic “find Space Invaders march / shot / hit /
UFO source sounds” gap for Windigo intake.

It means the remaining audio hunt can be narrower and more explicit:

- missing player-loss / death sound truth
- missing start / credit / insert-coin truth
- live gameplay windows where these preserved cues are heard in context

## What It Does Not Replace

This pack does not replace:

- preserved gameplay for march tempo, bunker wear, descent pressure, and
  timing-in-context
- title / attract / score-strip stills
- score-table and rules sources
- gameplay windows that show how these sounds layer during active play
