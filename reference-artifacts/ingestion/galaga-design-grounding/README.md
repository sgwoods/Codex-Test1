# Galaga Design Grounding

This directory stores derived, human-reviewable source notes from online
Galaga references. It is not a raw scrape. The intent is to preserve stable
design assertions that can guide conformance scoring while keeping movement,
audio, and pixel claims grounded in measured media artifacts.

Current artifacts:

- `official-history-notes-0.1.json`: official Bandai Namco Galaga identity,
  rules, enemy-role, and player-facing design notes.
- `player-guide-notes-0.1.json`: player-experience notes from expert-guide
  sources, especially entrance-pattern anticipation, challenge-stage
  learnability, and capture/rescue meaning.
